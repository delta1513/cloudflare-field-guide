# CloudFlare Field Guide

**[Take a Look](https://delta1513.github.io/cloudflare-field-guide/) 👀**

This is a quick idea that came to my head recently. I've been dealing more and more with CloudFlare at work, and I wanted to learn about what products they offer without any of the dryness of their technical documentation, and also without the whimsy and abstractions of their marketing materials. Ideally, I wanted to understand quickly:
- What they offer
- What it does
- How it works from a high-level
- How people use it
- And where I can learn more

# How to Maintain

There is a skill in the .claude/ directory that you can point an AI agent to. I used this to create each article.
